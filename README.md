# Personal Portfolio Website

A modern and responsive personal portfolio website built using pure HTML, CSS, and Vanilla JavaScript. This project showcases my skills, projects, and contact information in a clean and interactive way.

## 🚀 Overview

This portfolio is designed to represent my work as a Front-End Developer. It focuses on simplicity, performance, and user experience without relying on frameworks.

The website includes essential sections like:
- Home / Hero section
- About Me
- Skills
- Projects
- Contact

## 🎯 Project Goals

- Build a fully functional portfolio using Vanilla JavaScript
- Practice DOM manipulation and event handling
- Create a responsive design for all screen sizes
- Improve UI structure and code organization
- Showcase personal projects in a clean layout

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript (ES6+)

## ✨ Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Smooth scrolling navigation
- Interactive UI elements
- Dynamic content rendering using JavaScript
- Clean and minimal design
- Organized and readable code structure
